Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The title element defines the title for a webpage and is important for SEO.
  • The style element defines the CSS styles used for a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata, such as the character set, description, keywords, author and viewpoint.
  • The script element embeds client side scripts into the HTML document.
  • The body element represents the visible content shown to the user.
  • The body element is an example of a semantic element. Semantic elements have an important part to play in user accessibility.
  • The div tag is used to create a division in the page.
  • The span tag is used to create an inline division.
  • The p tag is used to start a new paragraph.
  • The heading tags (h1 - h6) are sized from largest to smallest.
  • The ul tag is used to display an unordered list with bullet points.
  • The ol tag is used to display an ordered list with numbers.
  • The img tag is used to insert an image to display on the webpage. It contains src and alt attributes.
  • The src attribute contains the location of the image or file, either a website URL or from your assets folder.
  • The alt attribute contains the description of the image, which is useful for individuals using screen readers and SEO optimisaion.
  • The br element creates a line break within two blocks of content
  • The a element contains links to another website or page and contains the href attribute.
  • The href attribute contains the URL to link another webpage.
  • The footer element is used is used at the bottom of the webpage and contains information such as author, copyright, contact, sitemap and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The link href element allows us to link our style file to our html file, which goes after the final meta tag
  • The * selector allows us to apply the css styles to all elements on the page
  • The class selector allows us to make changes to select elements on a page

Git

  • git status: check what branch currently in and if your files are staged.
  • git checkout -b branch-name: create new branch and switch to it.
  • git clone git@github.com:USER-NAME/REPOSITORY-NAME.git: clone repository from GitHub onto computer.
  • git add: add files to staging area.
  • git commit -m "Insert commit message": commit changes to files with commit description.
  • git log: contain information for entries including author and date/time.
  • code .: open all files within directory.
  • ls: view files within current directory.
  • pwd: display current directory location.
  • cd: navigate within the command line.
  • mkdir: create new directory.
